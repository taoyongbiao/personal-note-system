# 模板使用

## 模板位置

所有笔记模板都存储在 `daynote/model/` 目录中，包括：

- `灵感.md` - 灵感笔记模板
- `日报.md` - 日报模板
- 其他自定义类型模板

## 模板结构

### 灵感笔记模板

```markdown
# 灵感笔记 - {{date}}

## 创意点
- 

## 遇到的问题
- 

## 解决方案
- 

## 学到的知识
- 
```

### 日报模板

```markdown
# 日报 - {{date}}

## 今日完成
- 

## 今日遇到的问题
- 

## 明日计划
- 

## 学到的新知识
- 

## 其他事项
- 
```

## 自定义模板

您可以创建自己的模板：

1. 在 `daynote/model/` 目录中创建新的 [.md](file://c:\Users\P30015874206\Desktop\note\README.md) 文件
2. 使用 `{{date}}` 作为日期占位符
3. 设计您需要的章节结构
4. 通过命令行使用新模板：
   ```bash
   python scripts/create_note.py --template [模板名]
   ```

## 模板变量

### 日期变量
- `{{date}}` - 当前日期（格式：YYYY-MM-DD）

### 创建时间变量
- 时间戳会自动添加到文件名中

## 模板最佳实践

1. **保持一致性** - 使用相同的结构和格式
2. **预留空间** - 为内容预留足够的空间
3. **清晰标题** - 使用清晰的章节标题
4. **适当留白** - 合理使用空行和分隔符
