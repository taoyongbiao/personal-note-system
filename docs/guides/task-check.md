# 任务检查

## 功能概述

主线任务检查机制用于检查并更新项目中的主线任务状态。该功能可以帮助您：

- 跟踪项目中的主要任务进展
- 检查任务完成情况
- 更新任务状态

## 主要文件

- [main_check.py](file://c:\Users\P30015874206\Desktop\note\main_check.py) - 主线任务检查的主要逻辑
- [task.md](file://c:\Users\P30015874206\Desktop\note\task.md) - 任务定义文件

## 使用方法

### 运行任务检查

```bash
python main_check.py
```

### 通过VSCode任务运行

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "Tasks: Run Task"
3. 选择相应的任务检查任务

## 任务定义

任务定义在 `task.md` 文件中，包含：

- 主要任务列表
- 任务状态（待开始/进行中/已完成）
- 任务优先级
- 任务截止日期

## 任务格式

任务文件使用以下格式：

```markdown
# 项目任务规划

## 主要任务

### 高优先级
- [ ] 任务1 - 描述
- [x] 任务2 - 描述（已完成）

### 中优先级
- [ ] 任务3 - 描述
- [-] 任务4 - 描述（进行中）

### 低优先级
- [ ] 任务5 - 描述
```

## 状态标识

- `[ ]` - 待开始
- `[-]` - 进行中
- `[x]` - 已完成
