# 个人笔记系统文档

欢迎来到个人笔记系统文档中心！这是一个基于 VSCode 的个人笔记管理系统，旨在帮助您高效地记录日常工作、自动生成日报以及评估工作状态。

## 系统概览

个人笔记系统是一个综合性的知识管理工具，具有以下核心功能：

- **自动创建日期目录** - 系统会自动创建以 `YYYYMMDD` 格式命名的子目录
- **模板化笔记创建** - 通过预设模板快速创建标准化的笔记文件
- **快捷键操作** - 提供便捷的快捷键快速创建各类笔记
- **AI驱动的笔记评价** - 使用AI对笔记进行自动总结和评价
- **主线任务检查** - 检查并更新项目中的主线任务状态
