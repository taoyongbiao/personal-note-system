# 功能特性

## 核心功能

### 自动创建日期目录
- 系统会在需要时自动创建以 `YYYYMMDD` 格式命名的子目录，例如 `20251222`
- 无需手动创建目录，简化文件组织结构

### 模板化笔记创建
- 通过预设模板快速创建标准化的笔记文件
- 支持多种笔记类型：灵感笔记、日报、会议记录、技术调研等
- 自动替换模板中的 `{{date}}` 占位符为当前日期

### 快捷键操作
- 提供快捷键快速创建各类笔记
- 支持命令面板操作作为备选方案
- 无需手动操作，提升工作效率

### AI驱动的笔记评价
- 自动扫描 `daynote` 目录下的今日笔记
- 使用AI对笔记进行总结和评价
- 生成详细的评价报告

### 主线任务检查
- 检查并更新项目中的主线任务状态
- 提供任务完成情况的可视化

## 使用场景

### 日常工作记录
- 记录日常工作内容和进展
- 保存灵感和想法
- 跟踪问题解决过程

### 知识管理
- 系统化保存学习笔记
- 整理技术调研结果
- 归档会议记录

### 项目跟踪
- 记录项目进展和决策
- 跟踪任务完成情况
- 评估工作状态
