# 项目介绍

## 项目背景

个人笔记系统是一个基于 VSCode 的个人笔记管理系统，旨在帮助用户高效地记录日常工作、自动生成日报以及评估工作状态。该系统通过模板化笔记创建、AI驱动的笔记评价等功能，提升个人知识管理效率。

## 项目目标

- 提供便捷的笔记创建机制，支持多种笔记类型
- 实现自动化的笔记整理和归档
- 利用AI技术对笔记内容进行智能分析和评价
- 通过主线任务检查机制，帮助用户跟踪项目进展

## 核心价值

- **提高效率**：通过模板化和自动化减少重复操作
- **知识沉淀**：结构化存储个人工作和思考
- **智能分析**：利用AI技术对笔记进行总结和评价
- **状态跟踪**：通过任务检查机制跟踪项目状态
